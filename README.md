# CSC 324 Portfolio

This repository contains my CSC 324 portfolio website. It includes competency reflections, project artifacts, screenshots, and written explanations of my work throughout the course.

## Portfolio Organization

The items at the top of the portfolio are the competencies currently waiting for grading. These include the most recent competency submissions that I would like reviewed first.

After those entries, the portfolio continues with my project journal log. The journal documents my development process, project decisions, implementation notes, and reflections over time.

After the project journal, the remaining entries are competencies that were discussed and passed during 1-on-1 meetings. These are included for completeness and to show the full progression of my work throughout the course.
